## 📖 Materi

 * ***Ascending Sort***
* Algoritma pengurutan data dari nilai terkecil ke nilai terbesar.
* Contoh:
```
2 5 7 10 15
```

* ***Descending Sort***
* Algoritma pengurutan data dari nilai terbesar ke nilai terkecil.
* Contoh:
```
15 10 7 5 2
```

* ***Binary Search***
* Metode pencarian data pada array yang sudah terurut dengan cara membagi area pencarian menjadi dua bagian hingga data ditemukan.
* **Keunggulan:**
- Cepat dan efisien.
- Kompleksitas O(log n).

* ***Linear Search***
* Metode pencarian data dengan memeriksa setiap elemen satu per satu hingga data ditemukan.
* **Keunggulan:**
- Mudah diimplementasikan.
- Tidak memerlukan data yang terurut.
